						Welcome. 

		This document briefly outlines contents of the two folders 'DIY' and 'Ready'

------------------------------------------------------------------------------------------------------

'DIY' contains only the C source code for the interpreter and three sample input text files.
 

'Ready'	contains the files found in 'DIY', as well as compiled executables. The files are:

	BFInterpreter.c			C source code for the interpreter
	
	compiledBFInterpreter.exe	GCC compiled executable from BFInterpreter.c
	
	helloworld.out.c		generated from the compiledBFInterpreter.exe using the
					input argument values
					-i helloworld.in -o helloworld.out.c
	
	compiledHelloworld.exe		GCC compiled executable from helloworld.out.c
